MySQL
-------------------------------
Install MySQL Server
    Windows: https://dev.mysql.com/downloads/windows/installer/5.7.html
    Linux: (Debian) apt-get install mysql-server
    OSX: https://dev.mysql.com/downloads/mysql/ (select Mac OSX from dropdown)
Download Python MySQL Connector: https://dev.mysql.com/downloads/connector/python/2.1.html
